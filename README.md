# 📌 Bulletin Board

A virtual cork board you can use right in your browser.

🌐 **Live site: [https://mr-pythoneer.github.io/bulletin-board/](https://mr-pythoneer.github.io/bulletin-board/)**

## Features

- 📝 Sticky notes (drag, resize, recolour)
- 🧵 String connections between items
- ✏️ Freehand drawing & eraser
- 📁 File import — PDFs, text, CSV, code files
- 🎬 Video import with thumbnail preview
- 🖼 Image cards
- 🌐 Website cards (opens in embedded viewer)
- 📎 Tape strips
- 𝐓 Text labels
- Right-click any item for options
